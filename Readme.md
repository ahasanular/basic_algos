# DS-AL-101

This repo is for the all the seraching and sorting algorthm.

## Create a virtual environment
```bash
python3.9 -m venv env
```
## Activate the environment
For windows
```bash
env\Scripts\activate
```
For Linux
```bash
source env/bin/activate
```

## Run The project
```bash
cd src
```
```bash
python main.py
```
